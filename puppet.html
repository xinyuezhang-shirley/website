About

This project is an attempt to self-learn robot modeling using the MuJoCo physics engine. It is still a work in progress. The current demonstration shows the behavior of the puppet under different gravity contraints.

This project recreates a basic model of a puppet on strings using basic shapes and textures on MuJoCo.
