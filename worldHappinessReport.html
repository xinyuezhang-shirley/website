About

Inspired by the World Happiness Report developed by the Sustainable Devlopment Solutions Network, this simple applet provides the overall and subcategory rankings and scores of countries based on the 2022 report results. It utilizes a dropdown as user input, and in return provides rankings and scores via simple java script.

This project recreates my highschool CS project using self-taught html/js/css elements and Bootstrap. The original project was developed for my AP Computer Science Principles final project, and was originally coded using Code.org. Attached below is a video of my original AP Computer Science Principles project submission, which utilizes the 2019 report data results.
